也可以循环结束之后再更新答案

```python
class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        n,res = len(nums),inf
        cur = 0
        j = 0
        for i in range(n):
            cur += nums[i]
            while j <= i and cur >= target:
                res = min(res, i - j +1)
                cur -= nums[j]
                j += 1

        return res if res <= n else 0
```
