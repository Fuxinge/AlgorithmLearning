问题所在：while j < n and i >= 0 这个条件有点危险。更好的思考方式是：我们要在 i 到达最左端（0）且 j 到达最右端（n - 1）时停止。

```python
class Solution:
    def maximumScore(self, nums: List[int], k: int) -> int:
        i = j = k
        n = len(nums)
        res = nums[k]
        item = nums[k]
        while j < n - 1 or i > 0 :
            if j == n - 1 or (i and nums[i - 1] > nums[j + 1]):
                i -= 1
                item = min(item,nums[i])
            else :
                j += 1
                item = min(item,nums[j])
            h = j - i + 1
            res = max(res, h * item)

        return res
```
