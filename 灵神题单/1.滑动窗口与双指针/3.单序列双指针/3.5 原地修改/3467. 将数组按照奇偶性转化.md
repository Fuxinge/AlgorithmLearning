```python
class Solution:
    def transformArray(self, nums: List[int]) -> List[int]:
        cnt = Counter(x % 2 for x in nums)
        return [0] * cnt[0] + [1] * cnt[1]
```

双指针做法

```python
class Solution:
    def transformArray(self, nums: List[int]) -> List[int]:
        # 双指针
        n = len(nums)
        i,j = 0, n - 1
        while i <= j :
            if nums[i] % 2 == 0:
                nums[i] = 0
                i += 1
            elif nums[j] % 2 == 1:
                nums[j] = 1
                j -= 1
            else:
                nums[i],nums[j] = 0,1
                j -= 1
                i += 1

        return nums

```
