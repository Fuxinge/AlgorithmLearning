朴素思路

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n = len(nums)
        left = -1
        cnt = 0
        dummy = int(1e6)
        res = 0
        for right,c in enumerate(nums):
            if nums[right] != dummy:
                dummy = nums[right]
                cnt = 1
                left += 1
                nums[left] = nums[right]
            elif nums[right] == dummy:
                if cnt >= 2:
                    continue
                cnt += 1
                left += 1
                nums[left] = nums[right]

        return left + 1
```

栈的思路

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n = len(nums)
        st = 2
        if n < 2:
            return n

        for i in range(2,n):
            if nums[i] != nums[st - 2]:
                nums[st] = nums[i]
                st += 1

        return min(n,st)
```
