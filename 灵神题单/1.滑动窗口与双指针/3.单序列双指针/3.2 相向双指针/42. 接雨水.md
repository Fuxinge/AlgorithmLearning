双指针做法

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        res = pre = suf = 0
        i,j = 0,n - 1
        while i < j:
            pre = max(pre,height[i])
            suf = max(suf,height[j])
            if pre < suf:
                res += pre - height[i]
                i += 1
            else :
                res += suf - height[j]
                j -= 1

        return res
```
