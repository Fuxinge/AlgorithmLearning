k == n的时候要做一个特判

```python
class Solution:
    def maxScore(self, nums: List[int], k: int) -> int:
        # 反着考虑就是n - k大小的最小窗口和
        s = sum(nums)
        n = len(nums)
        if k ==n :
            return s
        k = n - k
        res = inf
        temp = 0
        for i in range(n):
            temp += nums[i]
            if i < k - 1:
                continue
            res = min(res,temp)

            temp -= nums[i - k + 1]

        return s - res
```
