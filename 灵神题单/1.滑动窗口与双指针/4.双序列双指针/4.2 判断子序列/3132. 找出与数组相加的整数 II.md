## 灵神思路

由于只能移除两个元素，所以 `nums1` 排序后的前三个最小值中，必然至少有一个会被保留下来。

这意味着，我们只需要枚举保留下来的最小元素是：

- `nums1[0]`
- `nums1[1]`
- `nums1[2]`

注意：保留下来的最小元素不可能是 `nums1[3]` 或更靠后的数。

因为如果最小保留值从 `nums1[3]` 开始，就说明 `nums1[0]`、`nums1[1]`、`nums1[2]` 都被删掉了，但题目只允许移除两个元素。

例如，排序后：

- `nums1 = [2,5,6,7,8,10]`
- `nums2 = [3,4,5,8]`

如果假设 `nums1` 中保留下来的最小元素是 `nums1[1] = 5`，那么：

`x = nums2[0] - nums1[1] = 3 - 5 = -2`

把 `nums1` 的每个数都加上 `x = -2`，得到：

`nums1' = [0,3,4,5,6,8]`

问题就转化成：判断 `nums2` 是否是 `nums1'` 的子序列。

如果是子序列，就说明剩下不匹配的两个数正好可以被移除。

```python
class Solution:
    def minimumAddedInteger(self, nums1: List[int], nums2: List[int]) -> int:
        n,m = len(nums1),len(nums2)
        nums1.sort()
        nums2.sort()

        for i in range(2,0,-1):
            x = nums2[0] - nums1[i]
            j = 0
            for v in nums1[i:]:
                if nums2[j] == v + x:
                    j += 1
                    if j == m :
                        return x

        return nums2[0] - nums1[0] # 保证有答案

```
