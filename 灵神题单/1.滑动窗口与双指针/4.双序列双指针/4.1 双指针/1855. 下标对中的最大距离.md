```python
class Solution:
    def maxDistance(self, nums1: List[int], nums2: List[int]) -> int:
        n,m,i ,j = len(nums1),len(nums2),0,0

        res = 0
        while i < n and j < m:
            if nums1[i] <= nums2[j]: # i <= j 不能写这个条件
                res = max(res,j - i)
                j += 1
            else :
                i += 1

        return res
```

# 注释的理由

假设某一步遇到了 nums1[i] > nums2[j]。

代码进入 else，执行 i += 1。

假设执行完后，i 跑到了 j 的前面，也就是 i > j 了。

下一次循环时，就算此时 nums1[i] <= nums2[j] 已经成立了，但因为你写了 i <= j and ...，整个 if 还是会返回 False！

于是代码再次进入 else，i 继续 +1。

从此以后，i 永远大于 j，代码永远走 else，i 一路狂奔直到越界，而 j 永远停在原地，再也没有机会往前走了。
