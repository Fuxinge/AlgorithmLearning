排序+双指针
如果把j = 0 放到while前一行，不对nums1排序，似乎会退化时间复杂度到O(n^2)

```python
class Solution:
    def findTheDistanceValue(self, nums1: List[int], nums2: List[int], d: int) -> int:
        n,m = len(nums1),len(nums2)
        j = 0
        nums1.sort()
        nums2.sort()
        res = 0
        print(nums2)
        for c in nums1:
            while j < m and nums2[j] < c - d:
                j += 1
            if j == m or nums2[j] > c + d:
                res += 1
        return res
```
