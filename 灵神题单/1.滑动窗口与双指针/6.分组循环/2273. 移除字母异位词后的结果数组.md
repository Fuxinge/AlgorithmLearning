pairwise做法

```python
class Solution:
    def removeAnagrams(self, words: List[str]) -> List[str]:
        n = len(words)
        idx = 1
        for s,t in pairwise(words):
            if Counter(s) != Counter(t):
                words[idx] = t
                idx += 1
        del words[idx :]
        return words
```

分组循环做法

```python
class Solution:
    def removeAnagrams(self, words: List[str]) -> List[str]:
        n = len(words)

        res = []

        i = 0

        while i < n:
            start = i
            i += 1

            while i < n and Counter(words[i]) == Counter(words[i - 1]):
                i += 1
            res.append(words[start]) # 直接赋值到words应该也可以

        return res

```
