也可用哈希表，一次遍历，类似两数之和，空间复杂度高

```python
class Solution:
    def arithmeticTriplets(self, nums: List[int], diff: int) -> int:
        n = len(nums)
        res, i, j = 0, 0, 1
        for x in nums:
            while nums[j] + diff < x:
                j += 1
            if nums[j] + diff > x:
                continue
            while nums[i] + 2 * diff < x:
                i += 1
            if nums[i] + 2 * diff == x:
                res += 1

        return res
```
